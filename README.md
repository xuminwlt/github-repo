# github-repo
github maven repo

Use this repo like

```
    <repositories>
        <repository>
            <id>maven-repo-github-releases</id>
            <url>https://raw.github.com/xuminwlt/github-repo/master/releases</url>
        </repository>
        <repository>
            <id>maven-repo-github-snapshots</id>
            <url>https://raw.github.com/xuminwlt/github-repo/master/snapshots</url>
        </repository>
    </repositories>
```
